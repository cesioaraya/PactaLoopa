# # 🤝 PactaLoopa

**PactaLoopa** es una aplicación web liviana para la gestión de ahorros comunitarios.

## 🚀 Características
- Configuración de monto y frecuencia.
- Sorteo/Asignación de turnos.
- Checklist de pagos en tiempo real.
- Interfaz optimizada para móviles (PWA).

## 🛠️ Tecnología
- **Lenguaje:** Python 3.9+
- **Framework:** Streamlit
- **Base de Datos:** Supabase (PostgreSQL)

## ⚖️ Disclaimer
PactaLoopa es una herramienta de registro. No procesa, recibe ni custodia dinero. 
Los acuerdos financieros son responsabilidad exclusiva de los participantes.
